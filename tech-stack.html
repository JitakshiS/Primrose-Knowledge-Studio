First — the Supabase question

Why Supabase, and why it won't pause this time

Supabase Pro, from launch

Supabase only pauses projects on the free tier, after 7 days of inactivity. That's what bit us on the earlier project, where we moved to Firebase to avoid it.

This one is a live paid product, so we put it on Supabase Pro (~CAD 35/mo). Pro never pauses. We build on free, then flip to Pro right before launch.

We pick Supabase over Firebase here because the data is relational — members linked to subscriptions, videos grouped by pillar — which fits Supabase's database cleanly. Firebase made sense last time because that project didn't justify a paid tier; this one does.

Pro tier never pauses Fits relational data Free tier pauses after 7 days idle
